# Healthcare Workforce Analysis in Central Java (2022)

## Overview

This project demonstrates the process of Exploratory Data Analysis (EDA) using healthcare workforce data from Central Java Province in 2022.

The analysis includes:

- Data Loading
- Data Cleaning
- Descriptive Statistics
- Duplicate Checking
- Missing Value Handling
- Data Aggregation
- Correlation Analysis
- Data Visualization

## Tools

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy

## Skills Demonstrated

- Data Cleaning
- Data Wrangling
- Exploratory Data Analysis (EDA)
- Data Visualization
- Statistical Analysis
- Python Programming

## Dataset

Healthcare Workforce Data in Central Java Province (2022)
